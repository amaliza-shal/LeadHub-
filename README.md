   How to Run Locally

1.  **Clone the repository**
    ```bash
    git clone https://github.com/amaliza-shal/LeadHub-.git
    cd LeadHub-
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the server**
    ```bash
    node server.js
    ```

4.  **Open in Browser**
    Visit `http://localhost:3000` to start learning!
