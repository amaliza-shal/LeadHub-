# LeadHub - Leadership Development Platform

## What This Project Does
LeadHub is a web platform for users to:

- Take professional leadership assessments
- Complete interactive leadership courses
- Practice skills through real-world challenges
- Track progress and development over time
- Earn XP and level up leadership skills

## How to Run

### Simple Setup
1. Download all project files to one folder
2. Double-click `index.html` to open in browser
3. Start using LeadHub immediately

### Required Files:
- `index.html` - Main application file
- `style.css` - All styling and design
- `script.js` - All functionality
- `Images.png/` - Background images folder

## Main Features
- **Leadership Assessments**: Style, Emotional Intelligence, Communication tests
- **Course System**: 12 leadership courses with progress tracking
- **Challenge Exercises**: 8 practical leadership scenarios
- **Progress Tracking**: XP system, levels, and achievement badges
- **User Profiles**: Personal development journey tracking

## What I Used to Build This
- **HTML** - Page structure and content
- **CSS** - Styling and responsive design
- **JavaScript** - All interactive functionality
- **Local Storage** - Save user data in browser

## How the Code Works
The app structure:

**Single HTML file (index.html)** with multiple sections:
- Login/Signup forms
- Home dashboard
- Courses section
- Challenges section
- Assessments section
- Progress tracking

**JavaScript (script.js)** handles:
- User authentication and sessions
- Assessment question flow and scoring
- Course progress tracking
- Challenge completion system
- XP and level calculations
- Local storage data management

**Navigation works by:**
- Showing/hiding sections with JavaScript
- No page reloads - smooth transitions
- URL hash updates for browser history

**Key features:**
- Assessments calculate scores and give personalized feedback
- Progress system tracks completion and awards XP
- All data saves locally in your browser
- Mobile-responsive design works on all devices
